# 3용스 - 성장 질문 웹 애플리케이션

5세, 8세, 11세 아이들을 위한 매일의 성장형 질문 시스템입니다.

## 특징

- **연령별 맞춤 질문**: 5세, 8세, 11세에 맞는 난이도로 질문 제공
- **3가지 카테고리**: 감정, 배움, 내일의 계획
- **다양한 답변 방식**: 텍스트, 음성, 선택형 답변 지원
- **성장 로그**: 시간이 지나면서 쌓이는 답변 데이터
- **성장 리포트**: AI 기반 감정 패턴 및 성장 인사이트 제공

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 사용 방법

1. 아이의 이름과 연령을 선택합니다
2. 매일 3가지 질문에 답변합니다
3. 답변은 자동으로 저장됩니다
4. 성장 로그에서 과거 답변을 확인할 수 있습니다
5. 성장 리포트에서 패턴과 인사이트를 확인할 수 있습니다

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Web Speech API (음성 인식)
- LocalStorage (데이터 저장)

## 프로젝트 구조

```
src/
  ├── components/      # React 컴포넌트
  ├── data/            # 질문 데이터
  ├── types/           # TypeScript 타입 정의
  └── utils/           # 유틸리티 함수
```
